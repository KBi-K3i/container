# 概要
個人的なコンテナファイル置き場です。<br>
`Hyper-V`に`Rocky Linux 10.1`を導入し、標準搭載されている`Podman`をランタイムとして使用しています。<br>
また、構成管理のために`Docker Compose`も導入しています。<br>

# 動作環境
上記は私個人の環境ですので、`Dockerfile`に対応しているコンテナランタイムであれば、`WSL`等でも動作します。

# Dockerfile について
ビルドを検証できるように、`requirements.txt`をコメントアウトしています。必要に応じて、コメントアウトを外してビルドしてください。

# docker-compose.yaml について
`Compose Spec v2`に基づいて、`Version`フィールド等は書いていませんが、導入しているバージョンに合わせ、必要に応じて修正してください。<br>
また、成果物をコンテナの外に出せるようにバインドマウントを行っていますが、こちらも必要に応じて修正してください。
